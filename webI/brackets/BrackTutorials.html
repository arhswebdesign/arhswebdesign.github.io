Submit the following:

  1. A multiple page website
  2. Creating Hyperlinks: to another website, to a place within the same web page, to an email program, to another webpage within the same website.
  3. Use graphics in the website
  4. Use a table in the site
  5. Create an external style sheet, use it to create the style for the site, and link web pages to it.

In the external style sheet set the code for At Least Four of the following:

  1. Background: image or color
  2. Text: color, alignment, background-color and line-height
  3. Font: family, size, style, and weight (boldness)
  4. Border: at least the bottom border
  5. Margin: left, right, top and bottom
  6. Padding: left, right, top and bottom
  7. Lists: set an image as the list-item marker
  8. Links (Pseudo-classes): set the color for these states of hyperlinks: link, visited, hover
