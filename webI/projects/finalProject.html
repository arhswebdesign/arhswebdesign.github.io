Project 15: Final Project

Code your final project.

You just did plans for this project in Project 14

Due date: Friday, 4/16/21

Keeping in mind our discussions of good web design, create a web site. This will be an example of your best work as web designers. Put all of the skills you have learned and developed this term into this project.


The site should contain at least 4 pages.


Requirements for this website

  • Complete the planning assignment before beginning the actual site construction.
  • Make a fully styled & designed template with external style sheet before building your first page. This saves typing, because you can start with a copy of this template for each of the pages in the site
  • Organize all files for this site into its own folder.
  • Create an images folder in the website folder and put all images in it.
  • Name the "home" page index.html.
  • Give each page a unique <title> tag.
  • Include alternate text for all images.
  • Navigation should be easily accessible and simple to manage.
  • Organize your html code so that it is easy to read.
  • Give all files meaningful names. For example: “Paintings” is meaningful, but “new page 4” is not. “Facts” is meaningful but “untitled” is not.
  • Use external style sheet/s to set the style for the website.
  • Use tables to help layout the pages. Laying out the pages with CSS is an alternative to using tables for layout.
  • Make sure all links work, and that relative pathnames for the links are used except for links to the internet.
  • Indicate on the home page who designed the site.
  • Comment the code thoroughly
  • Cite all work that you are using that is not yours. For example, you may quote information from another source, but you need to give a citation for that source to give the creator credit. This includes written and visual work.
  • Have someone else review your site as you are working on it to see what the audience response may be.
  • Check the site thoroughly before submitting it to be sure links work, words are spelled correctly, credit is given, etc.
  • Any questions? Please ask.
