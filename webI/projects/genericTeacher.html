Building a Generic Teacher Website

Assignment: Create a generic website for a high school teacher to be used as a resource for his/her classes. This should be a general teacher site. In other words it should be adaptable to any teacher. Due Date: Tuesday, 11/10/09


NOTE: You may choose to use this assignment to meet the community service requirement for this course. If that is the case, then you need to inform me ahead of time and have the teacher contact me with an email. The teacher will likely have his/her own requests for pages and links.


The site should have 5 pages: Home, Course Information, Homework, Resource Links, and a Calendar. The calendar should be a table set up as a generic calendar that can easily be changed depending on the month.


Additional requirements for the website:

  • The site should be defined as a local site
  • Name the first page index.html
  • Organize all files for this site into its own folder
  • Create an images folder in the website folder and put all images in it
  • Use tables to create the layout
  • Use external style sheets to style all pages in the site
  • Be sure all links work
  • Include alternate text for all images
  • Include an email link
  • Navigation should be easily accessible from all pages and simple to manage
  • Organize your html code so that it is easy to read
  • Include comments in your code to explain what each section is about
  • Images: at least one that is a link
  • Color
  • Create some minimal word docs that will stand in as homework assignments, etc so that it shows how a doc is set up through the site for downloading.
