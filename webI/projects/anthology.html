Create an Anthology

Assignment: Compile an anthology of works by one artist. Due Date: Tuesday, 5/29/18

It may be a collection of poems, plays, songs, films, paintings or books. Include some content. For example if it is books, then include some brief excerpts. Include at least four works. Use a separate web page for each. Include a home page (named index.html) which has an introduction with biographical information about the author. On each of the pages for individual works, describe the work and give the rationale for including the works. For example you might justify your choices with reviews or critical anaysis. Use a consistent navigation scheme. Make sure that any page can link directly to all of the other pages.
