CSS Lesson Grading Rubric

Category 1 point 2 points 3 points 4 points
HTML Requirements and Accuracy Student completes one or fewer of the pages and/or meets almost no requirements Student completes two of the pages and/or makes significant errors Student completes all three pages, some small errors made Student completes the internal, external, and internal/external pages with no errors, pages are linked together
CSS Requirements and Accuracy Less than 50% of the required styles are present and/or working 50% or more of the required styles are present and/or working 75% or more of the required styles are present and/or working All styles (background, text, font, border, margin, padding, list, pseudo-class, custom html element) are present and working, all three pages are linked to the same CSS file
Code Style Code style is inconsistent, hard to read, and results in errors in the code Code style is inconsistent but is somewhat readable, some errors present Code style is fairly consistent and readable, minimal errors are present Code style is consistent, readable, and has no errors
Indentation and Comments 1 of the 4 things at right is done 2 of the 4 things at right are done 3 of the 4 things at right are done Indentation of both html and CSS done according to the Granneman article. Code comments used in both html (2 or more) and css (2 or more).
Creativity No creativity present (Examples: titles not changed, lists contain the same text, the same color is used everywhere, etc.) "Bare minimum" creativity requirements met (Examples: titles changed, lists are unique, different colors used, etc.) Creativity goes beyond the minimum in some places (Examples: unique styles, appealing color scheme, custom images, interesting site content, etc.) Creativity consistently goes beyond the minimum throughout all parts of the website
