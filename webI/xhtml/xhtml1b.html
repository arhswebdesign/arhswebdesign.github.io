Styling html


Note: Do Evaluating Web Design Assignment Before Intro. Styling- see Due Dates Page.

1. Preparing to work:

Open Brackets, this web site, and the w3schools web site, and keep these all open throughout each class. You will be writing the html and CSS code in Brackets.

2. Saving your work:

In order to see the html code as a web page, you must save all html work with a .html file extension. For example, you could save a file as basic_tags.html .

Save all of your work to:

S:/HS/Student/Web Design/student_folders/yourName/ folder. This one and the first assignment should both go in the "early_work" subfolder. After this, you will start putting things in "middle_work."

3. Viewing your work:

There are two ways to view your html work as a web page,

  1. Use the lightning bolt in Brackets
  2. Type Windows-E and browse to your folder. Double click on your page. If it doesn't show the Chrome icon, then you might not have saved the file with a .html file extension. Try saving it again.

4. View your web page often:

Your working rhythm should be to write some html code, then check out the web page to see the new changes. Change and view. Change and view.

    • background color

    • text align
