1. Preparing to work:

Open Brackets, this web site, the w3schools web site and your web design folder, and keep these all open throughout each class while you are learning html. You will be writing the html code in Brackets.

2. Saving your work:

In order to see the html code as a web page, you must save all html work with a .html file extension. For example, you could save a file as basic_tags.html .

Save all of your work to:

S:/HS/Student/Web Design/student_folders/yourName/ folder. The early assignments should all go in the "early_work" subfolder.

3. Viewing your work:

There are two ways to view your html work as a web page,

  1. Use the lightening bolt in Brackets
  2. Type Windows-E and browse to your folder. Double click your page's file, which should appear with the web browser icon. If you are not seeing the web browser icon, then you must not have saved the file with a .html file extension. Try again.

4. View your web page often:

Your working rhythm should be to write some html code, then check out the web page to see the new changes. Change and view. Change and view.

Frequently Asked Questions

Q: After I have edited an html file, I cannot view the result in my browser, why?


A: Make sure that you have saved the file with a proper name and extension like "mypage.html". Also make sure that you use the same name when you open the file in your browser

Q: I have tried to edit an html file but the changes don't show in the browser. Why?


A: The browser caches your pages so it doesn't have to read the same page twice. When you have changed a page, the browser doesn't know that. Use the browsers refresh/reload button to force the browser to read the edited page.
