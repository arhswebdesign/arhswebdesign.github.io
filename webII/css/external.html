First page (CSS_with_external.html):

Create some organizational/layout structure using a layout table and different elements on the page. This should be new work. Put all of the style rules listed below into an external style sheet.

Second page (CSS_external_internal.html):

Create a second page. It can initially be the same as the first page. Change the title. It should link to the same external style sheet.

Using internal style, intentionally set up at least five conflicts. For example, you could style the background color salmon, even though it might already have been styled yellow in the external style sheet. This is so you can see for yourself how the style rules "cascade". In other words, which style is applied when 2 or more are giving different style directions?

Third page (CSS_all_three.html):

The third sheet should start off identical to the second. Then add inline style to change the same two (or more) style rules, yet again. So, for example, instead of yellow or salmon, you could make the background pink. See how the page looks.

When finished you should have 3 html pages all linked to one single css file.

 

In the style sheet, set the code for the following:

  • Background: image or color
  • Text: color, alignment, background-color and line-height
  • Font: family, size, style, variant and boldness (font-weight)
  • Border: at least style the bottom border
  • Margin: left, right, top and bottom
  • Padding: left, right, top and bottom
  • Lists: use at least three different lists: one ordered and one unordered list (set the list-item-marker style) and a third list where you set an image as the list-item marker (list-style image).
  • Pseudo-classes: set the color for these states of hyperlinks: link, visited, hover
  • Give at least one html element a class and style that class in the style sheet.
