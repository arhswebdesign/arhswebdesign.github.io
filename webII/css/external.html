Assignment Due Date: Wednesday, 2/5/20

First page (just_external.html):

Make a nice-looking web page that includes, among other things, text-containing elements (p, h2, h3, etc.) to which you have applied a background color or image that is different from that of the body. This should be all new code. Put all of the style rules into an external style sheet.

Include all of the following (at least):

  • Background: image or color
  • Text-containing elements: color, alignment, and background-color (or background-image)
  • Font: family (stack or use Google fonts), size, and boldness (font-weight)
  • Border border
  • Margin
  • Padding
  • Lists: use at least two different lists: one ordered and one unordered list
  • Pseudo-classes: set the color for these states of hyperlinks: link, visited, hover
  • Give at least one html element a class and style that class in the style sheet.

Second page (external_internal.html):

Create a second page. It can initially be a copy of the first page. Don't forget to change the title. It should continue link to the same external style sheet.

Don't change the external SS.

Using internal style, intentionally set up at least three conflicts. For example, you could style the background color DodgerBlue, even though it might already have been styled aqua in the external style sheet. This is so you can see for yourself how the style rules "cascade". In other words, which style is applied when 2 or more are giving different style directions?

Third page (all_three.html):

The third sheet should start off identical to the second. Then add inline style to change the same two style rules, yet again. So, for example, instead of DodgerBlue or aqua, you could make the background plum. See how the page looks.

When finished you should have 3 html pages all linked to one single css file.

Add links so each page links to the other two.

Put all the files for this assignment, including images, if any, in a separate folder.
