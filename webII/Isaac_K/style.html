Consistency

If your website has multiple pages, you want them all to have the same look and feel, so people know that they are on your site and not a different one. That's where external style comes in. Use external style to create that look and feel, because it will be the same on all the pages.

 

Styling your site

Link your template to your style sheet. Have the html page open in a web browser window so you can periodically save your code and refresh the browser page to see if the changes worked, and were what you wanted. It's best to save and refresh every time you make a change so that an error doesn't get lost in the new code. Use margins and paddings to position elements and text on the page. If you want to see how I coded this site or its style, right-click, choose 'view page source', and from there you can go through the link to the style sheet to see that. Using borders, margins, padding, color, and other style commands, you can go from this to this. When styling your page, you want to be aware of how colors go together. Text should be easy to read, so it shouldn't be too close to its background color. They can be the same color, but one should be a dark shade, and one a light shade, so that the text is easy to read.
