Working in Web Design: Review

To begin

  • Open My Computer on the computer desktop and then open S:\HS\Student\webDesign\F period
  • Look for a folder inside S:\HS\Student\webDesign\F period with your last name.
  • Create other folders inside the folder with your name. Name these as follows: general, projects, tutorials.
  • Use these folders to help you manage all of your web design files. Save your web design work in the appropriate folder throughout the term. Put all of the first few assignments in "general."
  • Create other folders as necessary.

The shared drive honor system, S:\

  • work in your folder only
  • good idea to back up your work to your H:\ once a week.
  • report any problems with S:\ asap.

Daily routines

  • Log into your computer as bell rings
  • Open Brackets, the class website, and your web design folder.
  • Keep these open throughout the class.
  • Turn off monitors during teacher talk (10-20 minutes)
  • As the trimester progresses, there is less and less teacher talk and more hands on work.
  • Students hands on study (35-45 minutes)
  • Working rhythm: write html code, save it as .html file, refresh browser page. Save, refresh, save, refresh, save refresh.

Naming your files:

  • When you save a web document, whether it is a page, an image, or an object, give it a name related to the assignment, a name that makes sense, in the correct format.
  • Name your assignments with short, descriptive file names.
  • For example, assignment 3 should be named something like external_style_sheets.html
  • Use the underscore character instead of spaces between words in a file name.

All first pages of a web site are named index. This is a general rule which is true for most web hosts.
