Due Date: wednesday, 4/26/16:

In this assignment you will begin to use the new html5 semantic elements, as well as the div and span tags to explore how they work. These tagsare used to group and structure a web document. Think about how elements on a web page can be grouped to create structure, and then the group can be styled. Rather than styling a lot of elements individually, we can now style a group of elements.Divisions and Spans

Until recently, many advanced web pages were mostly made up of <div> elements, and they were distinguished using classes or ids.

In html5, new elements have been introduced.  This saves typing when styling, but also helps automated search engines learn more about your page's content and structure.
