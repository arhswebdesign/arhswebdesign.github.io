The goal of this project is to design and build an online version of the student newspaper, The Graphic

This website should represent your best efforts to show your web design skills

Requirements

Planning the website

  • Audience explanation
  • powerpoint sketches of page design ideas

Clear design concept

  • thought given to design
  • design creates a sense of unity in the site
  • shows a clear look and feel

Write clear directions to teach other students how to use the website.

  • Create a "Read Me" text file or create another web page for users to access the general directions on how to use the site
  • Use code comments to explain to users where to put content, inlcuding headlines, stories and images

Layout

  • Now that you have learned the box model, do not go back to using tables to layout a web page. Instead, use your new knowledge of the box for layout. Keep building on that so you continue to learn.

CSS Styling

  • All styling through the external style sheet
  • Show an understanding of inheritance
  • Show an understanding of the box model and layout of webpages using CSS: this includes floating and efficient use of selectors
  • font family
  • font size
  • font style
  • font weight
  • line height
  • background
  • color
  • tables, td
  • links
  • divs, spans
  • everything that can be styled, to the best of your ability

Site demonstrates "don't make me think" qualities
