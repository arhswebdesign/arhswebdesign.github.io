Converting the Digital Portfolio to a Generic Student Portfolio

Converting your web design II work portfolio into a genric student work portfolio

  • Create a folder structure for this new portfolio to include a main folder, a folder for each major academic subject, an electives folder, and an images folder
  • Modify the template of each unique page in the portfolio to fit this folder structure:
    • Home page
    • A page for each major academic subject
    • An electives page
    • A personal interests page
  • Work with internal style while building the template and then set up that style in an external style sheet when you roll out the rest of the pages
  • Name the first page of student portfolio index.html
  • No tables used for layout
  • Use div, box model concepts and floats to create the layout
  • Remember: floated items must have a width set, unless they are images, which already have a width
  • Be smart (efficient) with classes
  • Think about how inheritance can play a role
  • Navigation to all major areas in the site should be available on every page.
  • Test all links.
  • Create alt text for all images
  • Be sure to effectively use the keywords and description meta tags.
  • Create a Read me file to explain how to use the portfolio
  • Include code comments
  • Validate the code
