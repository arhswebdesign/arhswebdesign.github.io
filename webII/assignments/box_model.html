The Box Model

The box model gives us a way to do layout that is more sophisticated than using a table.

The <div> tag defines a division or a section in an HTML document. The <div> element is often used as a container for other HTML elements. You will need to use it for this assignment. Here is a good online resource to review and learn about how to use div:

  • GeeksforGeeks on <div>

In this assignment you will learn the box model and play with it to get a solid understanding.

View the slideshow. After viewing the slides, follow the link to the Head First Lounge. Right-click, save to your S: drive, in your folder. As you do so, select "Save as type: web page complete." Open the page in Brackets. Make doubly sure you are working with your own copy, not the original.

See the boxes "Our guarantee" and "Weekly elixir specials?" Where are these styled? What kind of padding, borders and margins are used? What controls the position of the guarantee? Open the CSS page "lounge.css" in Brackets. Remove the id="guarantee" from the html of the p element containing the guarantee. Save and view the result. Put it back the way it was. Down in the place where #elixirs is styled, comment out the float like this: /*float: right;*/. Save and view the result. Put it back the way it was.

Make sure the words of the guarantee fit within the blu-green area. If not, fix things so that they do fit.

I'd rather see the rest of the text (I call it the "write-up") placed in the area between the guarantee and the "elixirs" box. See if you can make that happen.

Then get the three sections (guarantee, write-up and exilirs box) to line up at their tops.

Still confused? I find Mozilla's Box Model Page to be a nice, clear, explanation.
