    The assignment:

  • Create 3 html files with latin text
  • Save the files as box_model1.html, box_model2.html, box_model3.html
  • Think about each html element as a box, as presented in the slideshow
  • Give each html element a border or background color so that the box is easily seen.
  • Play with margins and padding
  • Create different sized and shaped boxes using margins and padding
  • Don't worry too much about positioning boxes at this point. Just give it your best shot. We will soon be learning how to float and position boxes, i.e., html elements on the page
