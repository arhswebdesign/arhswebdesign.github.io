The Box Model

The box model gives us a way to do layout that is more sophisticated that using a table.

The <div> tag defines a division or a section in an HTML document. The <div> element is often used as a container for other HTML elements. You will need to use it for this assignment. Here is a good online resource to review and learn about how to use div:

  • GeeksforGeeks on <div>

In this assignment you will learn the box model and play with it to get a solid understanding.

View the slideshow. After viewing the slides, follow the link to the Head First Lounge. Right-click, save to your S: drive, in your folder. As you do so, select "Save as type: web page complete." Open the page in Brackets. Make doubly sure you are working with your own copy, not the original.
