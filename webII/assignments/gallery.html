Beginning the Gallery

Creating your gallery page

  • Create a folder in your portfolio folder named gallery
  • Create a page in your gallery folder and name it index.html
  • On the class gallery page in the class portfolio folder, add your name to the list of students in alpha order
  • Create a hyperlink with your name from the class gallery page to your gallery page
  • Create a link from your portfolio home page, index.html, to the class gallery page.
  • Create a link from your portfolio home page to your gallery home page
