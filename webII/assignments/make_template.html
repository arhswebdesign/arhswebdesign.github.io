Make an HTML template

Tuesday, 1/28/20:

  1. Create an html template page.
  2. Learn how to use it: When starting a new html page, open the template, and then select file -> save as -> give it a new name and the file extension .html.
  3. Include all standard elements: doctype, html, head, title and body.
  4. By using a template, you will not have to write the required html page tags every time you start a new page.
