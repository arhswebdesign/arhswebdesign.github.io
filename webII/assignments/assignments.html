Web Design II: General Advice

In all assignments include:

  • Simple navigation
  • With simple navigation one can go to and from all pages included in the assignment
  • The back button is not considered part of a web page's navigation
  • Code comments: In every style sheet you make, include CSS comments indicating where inheritance is happening. This will encourage us to keep inheritance in mind and create more efficient style sheets as a result. Also include comments in all HTML pages. Indicating structural areas and organizing features in the HTML structure are good things to comment.
  • Keep in mind and practice the idea of inheritance in CSS. Inheritance saves time and effort. Inheritance is where CSS properties are passed down to descendant elements. Just like we inherit certain traits from our ancestors, CSS elements inherit from their ancestors.
  • Once you learn the box model, do not go back to using tables to layout a web page. Instead, use your new knowledge of the box for layout. Keep building on that so you continue to learn.
  • All html standard tags
  • Run every page of code through the code validator. Fix the errors.
  • Avoid deprecated (retired) tags
