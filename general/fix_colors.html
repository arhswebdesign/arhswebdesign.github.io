Open the Run Box (Press Win+R)
Type %appdata% then hit enter
Open the Brackets folder.
Select the brackets.json file.
Double-click the file; it should open in Brackets.
add the line: 


    "maxCodeHints": 148
You can put it after the other things, inside the "}". Then you also have to add a comma at the end of the previous line. You are not allowed to put a comma after the 148, though. Also, bear in mind that if Brackets can't read the json file on start up, autosave will be off and you will have to save manually.
