Remote access to Groupwise from any internet connection outside the school network

  1. First, you must create a Groupwise password
  2. Open Groupwise while logged on at school
  3. On the main menu bar, click Tools, Options, Security
  4. Select the Password tab
  5. Type in a password to use when accessing Groupwise remotely. It does not have to be the same as your network password, though it may be.
  6. Retype the password to confirm it.
  7. Click OK
  8. Browse to the Arps home page
  9. Click on Email Web Access
  10. At the Security Alert Dialog box click Yes
  11. At the Groupwise Web Access page type in your Amherst schools network username, ex. presleyE and then type your password
